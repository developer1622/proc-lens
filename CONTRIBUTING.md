# Contributing to proc-lens

We appreciate your interest in contributing to `proc-lens`! Kindly read the guidelines below to get started.

## Code of Conduct
Please follow our standard Code of Conduct to ensure a welcoming environment for all.

## How to Contribute
1. Fork the repository and create your branch from `main`.
2. Write clean code conforming to Go conventions.
3. Ensure unit tests pass and add new tests where appropriate.
4. Open a Pull Request detailing the changes and do the needful.

for any questions or queries, please open an issue or contact the maintainers.
